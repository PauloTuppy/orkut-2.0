# Orkut 2.0 - Sales Information

## Product Overview

Orkut 2.0 é a rede social moderna que combina:

### Features Principais

- **Comunidades**: Grupos temáticos com chat real-time (estilo Orkut)
- **Chat**: Integração com MSN Messenger (contatos, status online/offline)
- **Feed RSS**: Leitura inteligente com Gist Memory e resumos IA
- **Audio Rooms**: Salas de áudio para conversas (estilo Clubhouse)
- **P2P Sharing**: Compartilhamento de arquivos (estilo Napster)

### Preços

**Plano Gratuito (Free)**
- Comunidades ilimitadas
- Chat com até 50 contatos
- RSS Reader básico
- 1 Audio Room por mês

**Plano Pro ($9.90/mês)**
- Tudo do Free
- Chat com contatos ilimitados
- RSS Reader com Gist Memory
- Audio Rooms ilimitados
- Suporte via chat

**Plano Enterprise (Custom)**
- Tudo do Pro
- API access
- Custom branding
- Suporte dedicado
- SLA garantido

## Perguntas Frequentes

### Como funciona o Gist Memory?

Gist Memory é tecnologia de IA que lê documentos longos:
1. Quebra o documento em páginas
2. Cria resumo de cada página
3. Você faz perguntas
4. IA consulta apenas páginas relevantes
5. Resposta em menos de 1 segundo

### Posso usar Orkut 2.0 no celular?

Sim! Temos apps nativos para iOS e Android com todas as features.

### Como posso compartilhar arquivos?

Use a feature P2P Share - é como Napster, peer-to-peer sem servidor central.
